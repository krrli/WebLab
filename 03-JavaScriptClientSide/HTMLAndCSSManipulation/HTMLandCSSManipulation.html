Erzeugen Sie eine einfache Webseite in HTML die einen oder mehrere Texte und einen Button enthält.
Verändern Sie den Inhalt oder Aussehen der Webseite, wie folgt:





1. Verändern Sie den Text.
2. Verändern Sie den Hintergrund des Texts
3. Verändern Sie den Hintergrund des Texts, sodass bei jedem drücken eines Buttons eine zufällige Farbe gesetzt wird.
4. Old Style Animation: Erzeugen Sie eine JavaScript Funktion, die ein Element verblassen lässt, i.e. es wird langsam unsichtbar. Wenden Sie diese auf den Text an.
